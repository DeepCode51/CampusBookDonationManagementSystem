## 功能介绍（完整见运行截图）
普通用户：普通用户可以进行图书捐赠、修改或删除待捐赠书籍，并修改个人信息和密码。用户还可以查看图书捐赠信息、个人捐赠记录以及图书审核结果，以便参与和管理自己的捐赠活动。

管理员：管理员负责管理图书捐赠的整体流程，能够查看并管理用户列表和待审核图书，进行图书的审核和删除操作。此外，管理员还可以查看捐赠记录以及修改个人信息和密码，确保系统的高效运作和图书的顺利流通。
